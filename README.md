# Safety-Constrained Average-Reward Markov Decision Processes

This repository contains code and plots for analyzing safety in stochastic systems through constrained average-reward Markov decision processes (MDPs).

The primary objective is to demonstrate the relationship between safety and performance using an average-reward MDP framework. In this context, safety is imposed as a constraint, while performance serves as the optimization objective. This formulation facilitates the analysis of the trade-off between achieving specified goals and maintaining safety.

The repository includes examples that address reach-avoid problems, policy evaluation, and comparisons between safety and performance. The code is primarily used to generate numerical results and figures for the associated publication.

## Repository Structure

The 'Code and Plots' folder contains Julia and MATLAB code, numerical examples, and figures generated for this study.

## Main Topics

* Constrained average-reward Markov decision processes (MDPs)
* Safety-critical decision making
* Reach-avoid problems
* Safety-performance trade-offs
* Linear programming formulations for policy computation
